# Increased Particle Limit

**Increased Particle Limit** is a mod for **Zumbi Blocks 2** that simply increases the particle limit from 3 to 256.

## Installation

1. Make sure you have the latest version of **BepInEx 5** installed.
2. Download the latest version of the mod from the [releases](https://github.com/tbv001/ZB2.IPL/releases) page.
3. Copy the DLL into the `plugins` folder of your BepInEx installation:

```
Zumbi Blocks 2 Open Alpha\BepInEx\plugins\IncreasedParticleLimit.dll
```
